Le bouclier fonctionne un peu comme le temps : il va compter les messages envoyés dans un channel jusqu'à atteindre un seuil.

# Bouclier

**Trigger** : Command

**Usage :** `$bouclier`

> Le bot commence un compte de message pour recharger le bouclier.
